# SAE_INLA_Loreto

Small area estimation (SAE) of education level and wealth index for the 53 districts of Loreto, Peru, over the period 2010–2020. Models are fitted using INLA with spatial (SPDE, BYM2) and temporal (RW2) components at the PSU level.

## Outcomes

- **Education** (`hv109_recat`): proportion of household heads with at least completed primary education.
- **Wealth index** (`hv270_recat`): proportion of households in the bottom two wealth quintiles.

## Data sources

| File / folder | Description |
|---|---|
| `data/wi_edu_2010_2020.csv` | ENDES 2010–2020 household survey data for Loreto (generated by `00`) |
| `data/censo_2017/` | 2017 Census data and PCA scores (53 districts) |
| `data/aux_data/malaria_dist_total.csv` | Malaria incidence by district and year |
| `data/aux_data/malaria_dist_species.csv` | Malaria incidence by species |
| `data/aux_data/data_shapefiles/` | District boundaries and river shapefiles |
| `data/loreto_adj.graph` | INLA adjacency graph for Loreto districts |

## Scripts

| Script | Description |
|---|---|
| `00.raw_data_extraction.R` | Downloads ENDES microdata from INEI and processes 2007/2017 Census raw files |
| `01_SAE_INLA_I_part_modified.Rmd` | Direct estimates (design-based) for education and wealth index; merges with census PCA covariates |
| `02.SAE_INLA_EDU_method2.Rmd` | SAE models for education: BYM2 and SPDE with RW2 temporal trend, with and without PCA covariates |
| `03.SAE_INLA_WI_method2.Rmd` | SAE models for wealth index: binomial and beta-binomial, BYM2 and SPDE |
| `05.Aggregated_consistency.Rmd` | Aggregated consistency check — compares model-based annual prevalences against ENDES direct estimates |
| `06.biscale_malaria.Rmd` | Biscale maps combining SAE predictions with malaria incidence |
| `Supplementary_material.R` | Supplementary figures and tables |

## Functions (`_functions/functions.R`)

| Function | Description |
|---|---|
| `consulta_endes2` | Downloads and reads ENDES microdata modules from INEI. Updated version of [horaciochacon/ENDES.PE](https://github.com/horaciochacon/ENDES.PE) |
| `reconstruir_eta_spde` | Reconstructs the linear predictor η from INLA posterior samples for SPDE models |
| `resumen_anual_samples` | Computes aggregated annual prevalence with 95% credible intervals from posterior samples |
| `get_legend2` | Extracts the legend grob from a ggplot for use in composite figures |
